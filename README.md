# Password Manager
Password validator makes sure that your Password is safe against being hacked from hackers or comprimised. So the Application and their Users are secure from being Hacked.

### How does it work?:
When you run the programm, it asks you weather you want to Validate your Password or Generate one.

## Password Validator:

when entering "validate", you enter your Password you have and it will check if its secure or not.

---
#### How to Reproduce:
1. `validate`
2. `(ENTER YOUR PASSWORD)`
3. It will display if it's strong or weak

---

## Password Generator:

when entering "generate", you have to enter a Number from 8 - 60.
- If you enter below 8, it will be insecure and marked as weak password.
- If you enter above 60, it will be considered as too long and unreadable, since its hard to type if you would ever need to type it by yourself

---
#### How to Reproduce:
1. `generate`
2. `(ENTER A NUMBER FROM 8 TO 60)`
3. It will return you the generated Password
